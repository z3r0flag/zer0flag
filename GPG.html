GPG

Encrypt your internet communications!

GPG is a public-private key based encryption system. Everyone has a public key which can be shared and a private key which must be kept secret. A message encrypted with a public key can only be decrypted using the corresponding private key. GPG keys can also be used to digitally sign messages, that way the recipient can be assured that the message he/she recieved is legitimate.
